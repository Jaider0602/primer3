COMPETENCIAS OFIMATICA.

Microsoft Word, Excel, Access y HTML son herramientas y lenguajes que se utilizan para propósitos diferentes, pero todos pueden ser parte de un flujo de trabajo digital.


➥Word: Es un procesador de texto que sirve para crear y editar documentos como cartas, informes, currículums, etc. Permite formatear texto, insertar imágenes, tablas y otros elementos para dar formato al documento.

➥Excel: Es una hoja de cálculo que permite organizar y manipular datos numéricos y alfanuméricos. Se utiliza para realizar cálculos, crear gráficos y analizar datos.

➥Access: Es un sistema de gestión de bases de datos que permite crear y manipular bases de datos complejas, como listas de clientes, inventarios o cualquier tipo de información estructurada. Facilita la organización y consulta de grandes cantidades de datos.

➥HTML: Es un lenguaje de marcado utilizado para crear la estructura y el contenido de páginas web. No es un lenguaje de programación, sino un lenguaje para definir cómo se muestra el contenido en un navegador.



En resumen: Word, Excel y Access son aplicaciones de software dentro del paquete Microsoft Office, cada una con funciones específicas para procesamiento de textos, hojas de cálculo y bases de datos respectivamente. HTML es un lenguaje fundamental para la creación de páginas web.
